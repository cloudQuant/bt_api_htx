# HTX (Huobi)

HTX (Huobi) exchange plugin for bt_api plugin for `bt_api`.

## Features

- Spot and futures trading\n- HTX exchange integration

## Installation

```bash
pip install bt_api_htx
```

## Usage

```python
from bt_api_bt_api_htx import htx
# Use the exchange
```

## Architecture

The package follows the standard bt_api plugin structure with feeds, containers, and gateway adapters.
